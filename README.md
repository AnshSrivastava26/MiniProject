# LMS Frontend

A modern and responsive **Learning Management System (LMS) Frontend** built using **React.js** and **Tailwind CSS**.

This project includes student and educator interfaces with features like course browsing, course details, enrollments, video player, and educator dashboard.

---

## 🚀 Features

- Student Dashboard
- Educator Dashboard
- Course Listing Page
- Course Details Page
- My Enrollments Page
- Video Player Page
- Add Course Page
- Responsive Navbar and Footer
- Modern UI Design
- Routing with React Router

---

## 🛠️ Tech Stack

- **React.js**
- **Tailwind CSS**
- **React Router DOM**
- **Clerk Authentication**
- **Quill Editor**
- **React YouTube**
- **RC Progress**
- **Humanize Duration**

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/AnshSrivastava26/MiniProject.git
```

### 2️⃣ Go to the project folder

```bash
cd MiniProject
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start the development server

```bash
npm start
```

---

## 🌐 Run Locally

After starting the app, open:

```bash
http://localhost:3000
```
